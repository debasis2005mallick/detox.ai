# 🛡️ Detox.ai | Zero-Trust Privacy Layer
> Let the AI do the thinking. Keep your data locally.

Detox.ai is a client-side Chrome Extension that acts as a privacy middleware between users and AI endpoints (like Gemini or OpenAI). It detects sensitive Personally Identifiable Information (Aadhaar, PAN, Emails, etc.) and tokenizes it in volatile memory *before* the payload leaves your device. When the AI responds, Detox securely restores the data to human-readable form, proving that zero-trust AI interaction is possible without sacrificing context.
